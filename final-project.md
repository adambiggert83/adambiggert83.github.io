# Final Integrated Course Advising System

## Overview

This project began as a Course Advising System developed in CS300: Data Structures and Algorithms.

Throughout the CS499 Capstone course, the project was enhanced in three categories:

### Software Design and Engineering
- Object-oriented architecture
- Modular design
- Input validation

### Algorithms and Data Structures
- Graph-based prerequisite analysis
- Depth-First Search (DFS)
- Circular dependency detection

### Databases
- SQLite integration
- Relational database design
- Prepared SQL statements

## Skills Demonstrated

- C++
- Object-Oriented Programming
- Data Structures
- Graph Algorithms
- SQL
- SQLite
- Secure Coding
- Defensive Programming

# Software Design and Engineering Enhancement

## Overview

The original Course Advising System was developed in CS300: Data Structures and Algorithms. While the application successfully loaded and displayed course information, it was implemented as a single-file procedural application with limited separation of responsibilities.

This enhancement focused on improving the overall software architecture through object-oriented design principles, modularity, maintainability, and defensive programming practices.


## Skills Demonstrated

* C++
* Object-Oriented Programming
* Software Architecture
* Encapsulation
* Modular Design
* Input Validation
* Defensive Programming
* Error Handling


## Before Enhancement

The original application contained several limitations:

* All functionality existed in a single source file
* Global variables were used to store course data
* Limited input validation
* Tight coupling between user interface and business logic
* Minimal error handling
* Difficult to maintain and expand


## After Enhancement

The enhanced application was redesigned using a modular object-oriented architecture.

Major improvements include:

* Separation of responsibilities into multiple classes
* Removal of global variables
* Improved maintainability
* Better scalability
* Enhanced validation and error handling
* Cleaner code organization


## Key Improvements

### CourseManager Class

Responsible for managing all course-related operations.

### FileLoader Class

Handles file processing and CSV data import.

### MenuSystem Class

Separates user interface logic from application logic.

### Validation Improvements

Additional input validation was implemented to prevent invalid menu selections and improve user experience.

### Defensive Programming

Improved error handling and validation of course data increased reliability and robustness.


## Why This Enhancement Matters

Software engineering is about more than making code function correctly. It also requires creating solutions that are maintainable, scalable, and easy to understand.

This enhancement demonstrates my ability to evaluate an existing solution and improve its quality through professional software development practices.
